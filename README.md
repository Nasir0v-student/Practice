# kosmos
## Назначение проекта
Универсальный инструмент для бранирования билетов в кинотеатре.
## Функции проекта
* Учёт пользователей
* Продажа билетов
*Добавление фильмов
*Сеансы
*Занятость зала
## Состав команды
* Фронтенд - Насиров Роман*
* Бэкенд - Путятов Даниил*
* Путятов - разрабатывает интерфейс таблицы "Залы"
* Насиров - разрабатывает интерфей таблицы "Сеансы"
