# Космос
## Назначение проекта
Универсальный инструмент для удобного учёта фильмов и сеансов.
## Функции проекта
* Учёт фильмов
* Добавление фильмов
* Удаление фильмов
* Учёт Залов
* Добавление залов
* Удалние залов
## Состав команды
* Фронтенд - Насиров Роман*
* Бэкенд - Путятов Даниил*
## Процент выполнения
